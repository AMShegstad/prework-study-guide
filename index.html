CSS

  • Enter your CSS notes here.
    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.
    • A class for an html element is marked in css with a period "."
    • An id, like a class but for a unique element, is marked with a hashtag/pound sign "#"

Git

  • Enter your Git notes here.
    • git status: checks what branch we are currently on.
    • git checkout -b branch-name: creates a new branch and switches to it.
    • "git add -A" stages the changes
    • "git commit -m 'message'" commits the changes, with -m allowing for a message between quotation marks.
    • "git pull origin main" makes sure the branch is up to date
    • "git push origin main 'branch name'" pushes all of the changes in that branch.

JavaScript

  • Enter your JavaScript notes here.
    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
    • Lines of JavaScript code should end in a semicolon, like in Java.
